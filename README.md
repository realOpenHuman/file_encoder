# PDF 助记词加密工具

使用 AES-256-GCM 军用级加密和 BIP39 助记词保护 PDF 文件。

## 加密方法

- **AES-256-GCM**：认证加密，同时保证机密性和完整性
- **BIP39 助记词**：24 个英文单词作为密钥，便于记忆和备份
- **Base85 编码**：加密结果保存为cryp文件

## 使用方法
在终端中运行bin文件即可，无python环境依赖
加密

    输入 e 选择加密
    输入 n 生成新助记词（或 y 使用已有助记词）
    输入 PDF 文件路径
    保存生成的 .cryp 文件和 24 个助记词

解密

    输入 d 选择解密
    输入助记词
    输入 .cryp 文件路径
    恢复原始 PDF
安全提示

    助记词是唯一的恢复方式，请手写备份，勿存云端
    任何人获得助记词都能解密文件
    密文被篡改会导致解密失败（安全特性）
